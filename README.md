
Password Strength Checker
Overview
This JavaScript project implements a password strength checker, allowing users to evaluate the strength of their passwords based on various criteria. The project includes functions for checking the presence of special characters, numbers, and assessing password length.

Features
Special Character Check: Verifies if a password contains at least one special character from a predefined set.

Number Check: Ensures that a password includes at least one numerical digit.

Password Strength Assessment: Evaluates the overall strength of a password based on the presence of special characters, numbers, and length.

Usage
The user is prompted to enter a password.

The program checks the entered password against specified criteria.

The user receives feedback on the strength of their password.

How to Run
Clone the repository and open the index.html file in a web browser. Follow the prompts to input a password and receive the strength assessment.

Project Structure
index.html: HTML file containing the user interface and script inclusion.

script.js: JavaScript file with functions for password checking and strength assessment.
